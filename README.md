## Course "Programming in Python"

Репозиторий содержит учебные материалы и практические работы, выполненные в рамках курса «Программирование на Python».

### Структура репозитория

- /lab1 — сборник из 4 лабораторных работ (базовые и продвинутые концепции Python)

- /okx_volume_profiles — индивидуальный проект: интерактивный дашборд для анализа криптовалютного рынка

### Индивидуальный проект: OKX Real-time Dashboard

Интерактивная панель управления для сбора, анализа и визуализации профилей объема (volume profiles) в реальном времени

- Работа ведется с парами BTC-USDT и ETH-USDT биржи OKX

- Используется WebSocket API для получения реальной рыночной информации

- Применяется асинхронность для параллельной работы с потоковыми данными
